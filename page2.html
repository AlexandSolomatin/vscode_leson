Ссылки на страницы

Тут опишутся параметры ссылок на разные страницы

  1. a - парный тег для добавления ссылки на страницу
  2. href - ссылка на файл страницы. Пример оформления ниже
  3. target = '_blank' - открывает ссылку на новой странице
  4. Так же можно прописать не назнавине страницы, а атрибут картинки
  5. Главная страница
