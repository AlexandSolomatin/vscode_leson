Тут что-то про таблицы

  1. table - закрываемый
  2. tr - строка. тоже парный тег
  3. td - столбец. тоже парный тег
  4. align= - атрибут выравнивания для td
  5. border= - выделение таблицы
  6. width - задает размер таблицы в %
  7. height= - задает высоту таблицы в пикселях
  8. colspan="" - выранивание столбца
  9. rowspan="" - выранивание строки
